## 05_vue_frontend/ 
vue3+element plus，后端仍然是04_jsp_servlet_mvc项目下com\example\web\jsonTestServlet目录
主要功能：
        页面数据表单展示
        分页展示
        新增数据
        删除与批量删除数据
        修改数据
        条件查询

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
