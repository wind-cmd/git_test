## 修饰关键字
private:    私有的，只能被本类
            默认的，只能被本类  本包
protected:  受保护的，只能被本类    本包    子类
public:     公共的，可以被任何类访问

## this和super
this 代表当前对象的引用，用于区分成员变量和局部变量、调用本类的其他构造方法以及返回当前对象的引用。
super 代表父类对象的引用，用于访问父类的成员变量和方法、调用父类的构造方法。
super() 必须放在构造方法的第一行，用于调用父类的构造方法。
super() 和 this() 不能同时出现，因为它们都必须放在构造方法的第一行。
